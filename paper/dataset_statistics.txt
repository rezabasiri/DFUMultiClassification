MODALITY IMAGE CHARACTERISTICS
------------------------------
                        Depth RGB       Depth Map       Thermal Map
Resolution:             1280x720        1280x720        1080x1440
Encoding:               Natural RGB     Viridis cmap    Viridis cmap
Sensor FOV mask:        None            None            ~47% black border
Median BB crop size:    55x56 px        55x56 px        63x82 px
Median unique colors    1,136           78              1,136
  per crop:
Near-uniform crops:     ~0%             6.4%            0.3%

Depth maps encode distance-from-camera as a viridis colormap (confirmed
100% match to matplotlib viridis LUT). Crops contain 15.9x fewer unique
colors than RGB, with 6.4% of crops effectively a single solid color.
Thermal maps also use viridis but encode temperature, producing richer
gradients (14.6x more color diversity than depth maps).

Thermal pattern is biologically consistent: inflammatory wounds are
hottest (increased blood flow), remodeling wounds are coolest (reduced
metabolic activity).

TRAINING CONFIGURATION (from joint optimization)
-------------------------------------------------
Image input resolution:  128x128 pixels (resized from native with aspect-ratio preservation + zero padding)
Backbone:               DenseNet121 (ImageNet pretrained) for all image modalities
Fusion:                 Feature concatenation
Training:               Two-stage (500 epochs frozen backbone, 30 epochs 5% unfreeze)
Loss:                   Focal loss (gamma=2.0, frequency-based class weights)
Metadata:               Random Forest (300 trees, depth 10, k=80 mutual-information features)
Ensemble:               Simple average of 8 metadata-containing combinations
Generative aug:         SDXL 1.0, 15% probability, 1-5% mix, 144 cached images/phase

NOTES
-----
- Patient count: 268 in raw metadata, 233 after multimodal matching (35 patients had no matched image pairs)
- Multiple DFUs per image: Some patients presented with multiple wounds captured in the same image
- Matching loss: 4.5% of depth annotations lost during depth-thermal-metadata matching
- Class imbalance: ~5:1 ratio P:R in the 3-class labeling
- Depth maps have minimal standalone discriminative value (kappa 0.19); most useful as complementary signal
- Thermal maps carry meaningful temperature-based signal, especially for distinguishing remodeling from inflammatory/proliferative phases
- Image size 128x128 selected via joint optimization; 256x256 causes excessive upsampling artifacts from small bounding box crops
